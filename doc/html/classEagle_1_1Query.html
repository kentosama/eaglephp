◆ findAll()

Eagle\Query::findAll ( array  $args = [])

Retourne un tableau contenant plusieurs enregistrements de la table.

Paramètres
$argsest un tableau contenant différents arguments facultatifs pour préparer la requête. Les clés pouvant être utilisées sont 'select', 'contain', 'join', 'conditions', 'order' et 'limit :
select doit contenir un tableau ou une chaîne de caractères indiquant les champs à inclure (['user_id', 'user_nickname', 'user_password']).
Par défaut à '*'
contain doit contenir un tableau indiquant les associations de l'entité à inclure dans la requête (['Roles', 'Reviews']).
Par défaut à FALSE
join doit contenir un tableau indiquant toutes les jointures à inclure dans la requête.
Par défaut à FALSE
conditions doit contenir un tableau indiquant les conditions de la requête (['user_nickname = ' => 'demo']).
Par défaut à FALSE
order doit contenir un tableau avec le nom des différents champs pour ordonner la requête (['user_nickname' => 'DESC']).
Par défaut à FALSE
limit peut contenir un entier ou un tableau pour limiter le nombre d'enregistrements à récupérer avec un offset ([10, 5]).
Par défaut à FALSE
Renvoie
array Tableau contenant les entités et leurs champs sélectionnés. NULL si aucune entité trouvées.
Avertissement
Les fonctions de rappels beforeFind() et afterFind() sont appelées pour chaque requête.
Voir également

Eagle\Query::findList ( array  $args = [])
